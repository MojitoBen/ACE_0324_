# **方向辨識系統**
**需安裝darknet**    

## _**Dir_main.py :主程式**_        
  偵測上下方向的車流    
  部分參數Area、Data跟SQL_server修改後可運作    
  Data中可選COCO訓練權重可辨識的物件    
  Area可用Area_set.py手選框出    
## _**Dir_LRmode.py :主程式2**_    
  偵測左右方向的車流
  Area設定高>寬時建議使用
